## 機能説明・現状

`benches/wasm.rs` は `/wasm/*` 適用 vs 非適用のレイテンシ差計測のみ。以下が未計測:

- インスタンスプール枯渇時の挙動（プールサイズ超の並行リクエスト）。
- WASM fuel 消費量・RSS の自動計測。
- 外部バックエンドスタブを要する「HTTP コールあり」フィルタのベンチ。

## 改修内容

- HTTP コールを行うフィルタ用のスタブバックエンドを `tests/test_backends` に追加しベンチ化。
